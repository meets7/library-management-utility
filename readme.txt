1. Install will need Ruby on Rails framework 4.0 minimum
2. Install MySQL database.
3. Execute the library.sql script which is in the rails-app/db folder of the application.
4. (Assuming a UNIX system) Navigate to the rails-app folder, and use command bin\rails server to start the rails default server.
5. Open browser and navigate to localhost:3000 (check in port number in terminal output after above command).
